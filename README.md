# Rajiv A R | Personal Portfolio 🚀

A sleek, responsive, dark-mode personal portfolio website designed to showcase my skills, experience, and projects as a Computer Science Engineering student.

## 🌟 Features

* **Dark-Mode First Design:** A modern, high-contrast interface using a custom dark color palette.
* **Fully Responsive:** Fluid layouts using CSS Grid and Flexbox, complete with a mobile-friendly hamburger navigation menu.
* **Sticky Navigation:** Header remains accessible at the top of the screen with a frosted-glass blur effect.
* **Smooth Scrolling:** Inter-page links smoothly glide to their respective sections.

## 🛠️ Built With

* **HTML5:** Page structure and semantic elements.
* **CSS3:** Styling, custom variables, responsive media queries, and animations.
* **Vanilla JavaScript:** Lightweight DOM (Document Object Manipulation) for the mobile navigation toggle.
* **Google Fonts:** Utilizing *Inter* for readable sans-serif body text and *JetBrains Mono* for technical accents.
